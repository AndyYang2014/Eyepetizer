# Eyepetizer-Kotlin-MVVM

#####      开眼视频是一款精品短视频日报应用，该项目采用kotlin语言以及MVVM架构，借助已知的一些开眼接口写的一个仿《开眼App》，主要是为了学习Jetpack相关架构组件。
